# SFX Library 
## Below are [CC0](https://creativecommons.org/public-domain/cc0/) sounds that are built into fusion engine.

## Slap:

```python
main.sfx.SLAP()
```

## Splash:

```python
main.sfx.SPLASH()
```

## Slip:

```python
main.sfx.SLIP()
```
